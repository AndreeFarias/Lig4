# Game Lig-4 
Game made in Java

The files and folder this project:

* __Lig4\br\com\poli__: Interfaces and game
* __\Lig4\lig4__: Simple classes

Software:
```
Eclipse / VSCode
```

Build:
```
All files .java
```

Run the application:
```
Lig4.java
```
